#  Alarm Clock Application 

This project is a **multi-functional desktop time management application** developed using **Python and PyQt5**.  
It combines three essential tools in graphical interface:

-  **Alarm Clock**
-  **Countdown Timer**
-  **Stopwatch**

The application allowing users to easily switch between features and manage their time efficiently.

---

##  Features

###  Real-Time Clock
- Displays the current system time
- Shows the current date
- Updates automatically every second



###  Alarm Clock
- Set an alarm for a specific time
- Automatically checks the time every second
- Plays an alarm sound when the time is reached
- Snooze option (5 minutes)
- Ability to stop the alarm completely



###  Countdown Timer
- Set timer duration using hours and minutes
- Start, stop, and resume the countdown
- Displays remaining time accurately
- Plays a sound and shows a message when time is up



###  Stopwatch
- Start stopwatch with second-level precision
- Pause and resume functionality
- Reset stopwatch
- Displays elapsed time in `HH:MM:SS` format
